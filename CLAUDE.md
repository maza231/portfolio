# Project Rules

This project is a modern portfolio website.

## Tech Stack

* React / Next.js using App Router and TypeScript
* Tailwind CSS
* Mobile-first design
* Clean component structure

## Design Style

* Modern, premium, simple
* Strong hero section
* Clear CTA buttons
* Good spacing
* No generic AI-looking layout
* No random gradients unless they are useful
* Take inspiration from:

  * Conor O'Hollaren
  * Gabriel Valdivia
  * Pablo Sánchez

## Code Rules

* Use reusable components
* Keep code readable and maintainable
* Avoid unnecessary dependencies
* Make all pages responsive
* Optimize images
* Test layouts on mobile
* Keep the project structure clean and easy to navigate

## Business Goal

This website is a modern and effective portfolio designed to showcase work clearly and make it easy for people to get in contact.

The portfolio should feel professional, polished, and memorable without becoming overly complex or generic.
